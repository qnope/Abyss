# Task 9: Add kernel capture helper to `Game` model

## Summary

Add a method to `Game` that checks whether the volcanic kernel has been captured by a given player, by scanning Level 3 map cells.

## Implementation Steps

### 1. Edit `lib/domain/game/game.dart`

Add import (if not already present):
```dart
import '../map/cell_content_type.dart';
```

Add method:
```dart
bool isVolcanicKernelCapturedBy(String playerId) {
  final map = levels[3];
  if (map == null) return false;
  for (final cell in map.cells) {
    if (cell.content == CellContentType.volcanicKernel &&
        cell.collectedBy == playerId) {
      return true;
    }
  }
  return false;
}
```

The volcanic kernel capture state is tracked via `MapCell.collectedBy` — when the player captures the kernel, `collectedBy` is set to their player ID (same pattern as treasure collection but semantically representing capture).

## Dependencies

- Task 1: `CellContentType.volcanicKernel` must exist
- Task 4: `build_runner` must have been run
